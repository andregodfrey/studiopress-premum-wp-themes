Streamline Child Theme
http://www.studiopress.com/themes/streamline

INSTALL
1. Upload the Streamline child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Streamline theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

DYNAMIC CONTENT GALLERY
1. Download the latest version of the Dynamic Content Gallery plugin (http://www.studiograsshopper.ch/dynamic-content-gallery)
2. Upload the Dynamic Content Gallery plugin folder via FTP to your wp-content/plugins directory, and follow the directions from the docs page to install it. (http://www.studiograsshopper.ch/dynamic-content-gallery/documentation) The dimensions used for the Dynamic Content Gallery is 640x300.
