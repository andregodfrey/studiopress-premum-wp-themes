Outreach Child Theme
http://www.studiopress.com/themes/outreach

INSTALL
1. Upload the Outreach child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Outreach theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

SLIDESHOW
The slideshow that is underneath the header is using the WP-Cycle plugin which has been provided in the outreach/plugins/ folder. Upload the wp-cycle folder via FTP to your wp-content/plugins/ directory and then activate it. You will also need to configure the plugin by going to the Plugins > WP-Cycle page in your dashboard.

FEATURED IMAGES
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media. In addition, the Outreach child theme creates the following thumbnail images you'll see below, which are defined (and can be modified) in the functions.php file. These are the recommended thumbnail sizes that are used on the Outreach child theme demo site.

Slideshow - 910px by 290px
Homepage - 275px by 100px
Mini - 65px by 65px
